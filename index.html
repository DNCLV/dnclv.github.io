Project 2
Number Guessing Game

The Number Guessing Game is a game of 5 rounds where the user has to guess "The Number" based on a "lower" or "higher" clue, which is received after guessing. The program uses Math.random to random generate a number between 1 and 100, which the user has to guess within 5 tries. If failed to do so, the user will be enlightened with "The Number" after the fifth guess.
